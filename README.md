# README

## Sortlist Backend Challenge

Developer: Loren Hale

### Ruby version
* ruby 2.2.1
* rails 4.2.3


### Configuration

Standard Rails setup

`
$ bundle install
`
`
$ rake db:setup
`

### How to run the test suite

Run specs via command line:

`
$ rspec
`
and
`
$ rake jasmine
`

### How to run the app

`
$ rails server
`
